What is Gaussian Naive Bayes?

Gaussian Naive Bayes is a probabilistic machine learning algorithm that utilizes Data Mining Technique to make predictions based on the likelihood of each feature given the class label.

How Does It Work?

The algorithm calculates the probability of each feature given the class label using the Gaussian distribution, and then combines these probabilities to make a prediction.
